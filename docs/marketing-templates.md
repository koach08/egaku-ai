Hi HN, I built EGAKU AI — a platform that gives you access to 30+ AI generation models in one place: Veo 3 (video with audio), Grok, Flux Pro, Kling 3.0, Sora 2, and more.

The problem I'm solving: each AI model has its own platform, pricing, and API. If you want to compare Flux vs Grok vs GPT Image 2, you need three separate accounts. EGAKU puts them all in one interface with unified credits.

Features:
- Text-to-image (12 models including Flux Pro, Grok, Ideogram v3)
- Text-to-video with audio (Veo 3, Grok Video, Kling 3.0 4K)
- Image-to-video, Face Swap, Style Transfer, Lip Sync
- CivitAI integration (100K+ community models)

Tech stack: Next.js + FastAPI + Supabase + fal.ai

I'd love feedback on the UX and which features matter most to you.
