# CivitAI 投稿ガイド（EGAKU AI プロモ用）

## アカウント作成

1. https://civitai.com にアクセス
2. 右上「Sign In」→ Google / Discord / GitHub / Email で登録
3. プロフィール設定（Settings → Profile）:
   - Username: egaku-ai（または任意）
   - Bio: 下記をコピペ

```
AI image & video generator with 40+ tools.
Use CivitAI models directly from EGAKU AI.
Now with AI Music Generator and Movie Maker.
Free to start → egaku-ai.com
```

   - Links: https://egaku-ai.com を追加

## 投稿手順（画像）

1. 右上の「＋」ボタン → 「Post Images」
2. 画像をドラッグ＆ドロップ（複数可）
3. 各画像に以下を設定:

### Image Details（必須）

- **Resource Used**: 使用したモデルを選択
  - Flux Pro → 「Flux.1」で検索して選択
  - SDXL系 → 使用したチェックポイント名で検索
  - モデルが見つからなければスキップ可
- **Prompt**: 生成に使ったプロンプトを貼る
- **Negative Prompt**: あれば貼る

### Post Details

- **Title**: 任意（なくてもOK）
- **Description**: 下記テンプレをコピペして編集

```
Generated with EGAKU AI (https://egaku-ai.com)
Model: [使用モデル名]
Prompt: [プロンプト]

40+ AI tools for image, video & music. Free to start.
```

- **Tags**: 関連タグを追加
  - 例: `girl`, `portrait`, `anime`, `realistic`, `landscape` など内容に合ったもの
  - `egakuai` タグも追加しておくと後で検索しやすい

### NSFW設定

- 画像がNSFWの場合: 「Mature Content」トグルをON
- レベル選択: Suggestive / Mild Nudity / Nudity / Explicit から選択
- SFW画像はそのまま（デフォルトOFF）

4. 「Post」ボタンで投稿完了

## 投稿のコツ

### 何を投稿するか

- EGAKU AIのショーケースから高品質なものを選ぶ
- SFWとNSFW両方投稿する（CivitAIはNSFW活発）
- 1回5-10枚くらいまとめて投稿するのが効率的
- バリエーション（同プロンプトで複数結果）があると見栄えする

### 投稿頻度

- 最初の1週間: 毎日3-5枚
- その後: 週2-3回
- 一気に50枚投稿するよりコンスタントに出す方が露出が増える

### エンゲージメント

- 他の人の投稿にも「いいね」やコメントする
- コメントがついたら返信する
- 「どうやって作ったの？」→「EGAKU AI使ってる、egaku-ai.comで無料で試せる」

## 画像のダウンロード方法（EGAKU AIから）

1. https://egaku-ai.com にログイン
2. My Gallery → 投稿したい画像を選択
3. 画像を右クリック → 「名前を付けて画像を保存」
4. または Explore Gallery から良い作品を選ぶ

## 注意点

- CivitAIの利用規約に従う（児童ポルノは絶対禁止）
- 他人の著作物をそのまま投稿しない
- スパム的な連続投稿は避ける（アカウントBANリスク）
- 宣伝色を出しすぎない。あくまで「作品投稿」がメイン、リンクはプロフィールとキャプションにさりげなく
