## Rejection Strategies

If a directory rejects the submission:

1. **Too commercial** — Emphasize free tier and open access
2. **Adult content concerns** — Note the SFW-first positioning (NSFW is opt-in with legal compliance)
3. **Duplicate/similar** — Lead with unique features (Prompt Battle, regional pricing, Photo Booth)
4. **Incomplete** — Add more screenshots and demo video link
