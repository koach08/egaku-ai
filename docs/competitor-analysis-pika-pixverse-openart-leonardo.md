## 各プラットフォーム詳細

### Pika（pika.art）
- 最新: Pika 2.5
- 価格: 無料(80cr) / $8-10 / $28-35 / $76-95/月
- 強み: Pikaffects（バイラルVFX）、Pikaframes、iOS app
- EGAKUより高い

### PixVerse（pixverse.ai）
- 最新: V6
- 価格: 無料(50cr) / $8 / $24 / $40+/月
- 強み: マルチショット、C1キャラ一貫性、20+レンズ制御、音声同時生成
- EGAKUより高い

### OpenArt（openart.ai）
- 100+モデル対応
- 価格: 無料(40cr) / $7-14 / $14.50-29 / $28-56/月
- 強み: ワンクリックストーリー、LoRA訓練、キャンバスエディタ
- EGAKUと似た価格帯

### Leonardo AI（leonardo.ai）
- 最新: Phoenix 2.0 + Canva傘下
- 価格: 無料(150日次トークン) / $10 / $24 / $48/月
- 強み: リアルタイムキャンバス、3Dテクスチャ、テキスト描画、19M users
- EGAKUより高い
